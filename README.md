## How to install
- Unity -> Window -> Package Manager  
- Click "+" at the top left corner  
- Add package from git URL  
- Insert `https://github.com/HitScale-SG/com.unity.ide.vscodefork.git`  
- Add  
- Done

> **Important notice for migrations**  
> The package id is now `com.unity.ide.vscodefork`, covering VSCode-based forks (Antigravity, Cursor).  
> Remove any older `com.boxqkrtm.ide.cursor` or `com.hitscale.ide.antigravity` package before installing to avoid id conflicts.
